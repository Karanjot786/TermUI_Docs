# Hooks Overview

Every hook exported by `@termuijs/jsx`, grouped by what it does. Click a hook name to jump to its detail page where one exists.

## Installation

```bash
npm install @termuijs/jsx
```

All hooks are named exports:

```ts
import { useState, useInput, useAsync, useInterval } from '@termuijs/jsx'
```
